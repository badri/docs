A project can be deleted by clicking the "Delete" button in the project detail page.

![Project delete](/images/proj-delete.png)

You will need to confirm if you really need to delete the project.

![Project delete confirmation](/images/proj-delete-conf.png)

If your project contains one or more apps and/or services, you can't delete the project. You will be presented with a different message.

![Project delete has apps](/images/proj-has-apps.png)

<Note>
This is to facilitate graceful creation and destruction of resources in the nodes.
</Note>

 Once the project is deleted, you will get an acknowledgement message and will be redirected to the project listing page.

 ![Project delete ack](/images/proj-delete-ack.png)
