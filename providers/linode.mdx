
To add a Linode token, click on the providers link in the top menu.

![Menu with providers](/images/providers.png)

Click on "Add new Cloud Provider keys". It will open a dropdown with different cloud providers.
Select "Linode" from the options.

![Add new cloud provider](/images/new-provider.png)

Alternatively, you can directly [visit](https://platform.shapeblock.com/providers/new/linode/) the page.

The form will present you with an option to add a token.

![Linode form](/images/linode-form.png)

Give it a human-readable name which will help you refer it later.

#### Instructions for Linode token

The below instructions assume that you already have a Linode account created and you’re logged into it.
You can get a Linode token for your account from [here](https://cloud.linode.com/profile/tokens).
Make sure that the “read/write” scope for the token is checked for everything.

![Linode new token](/images/linode-token.png)

Copy over the token and paste it in the form above and click on "save".

You should have your provider listed on the [providers](https://platform.shapeblock.com/providers/) page after adding the keys successfully.
